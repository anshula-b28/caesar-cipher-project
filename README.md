# 🔐 Caesar Cipher Tool (Encryption + Attack Simulation)

## Overview

This project is a GUI-based application built using Python and Tkinter that demonstrates the **Caesar Cipher encryption technique** along with basic **cryptanalysis using brute-force attack**.

It allows users to encrypt and decrypt messages and also simulate how attackers can break weak encryption methods.

---

## Features

* Encrypt text using Caesar Cipher
* Decrypt encrypted messages
* Brute-force attack simulation (tries all 26 shifts)
* User-friendly GUI using Tkinter
* Real-time encryption and decryption

---

## Tech Stack

* **Language:** Python
* **GUI Library:** Tkinter

---

## How to Run

1. Clone the repository:

```
git clone https://github.com/anshula-b28/caesar-cipher-project.git
```

2. Navigate to the project folder:

```
cd caesar-cipher-project
```

3. Run the Python file:

```
python caesar_cipher_gui.py
```

---

## How It Works

### Encryption

* Each letter in the message is shifted by a fixed number (shift value).

### Decryption

* The encrypted message is shifted back using the same key.

### Brute Force Attack

* The program tries all possible 26 shifts to reveal the original message.
* Demonstrates how weak classical encryption can be broken.
