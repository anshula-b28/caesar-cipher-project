# Caesar Cipher Encryption and Decryption Using Python

This project implements the Caesar Cipher algorithm using Python.

The program allows users to encrypt and decrypt messages by shifting
letters in the alphabet using a chosen key.

## Features
- Encrypt messages
- Decrypt messages
- Simple Python implementation

## Technologies Used
- Python
- Classical Cryptography (Caesar Cipher)

## Caesar Cipher Encryption and Decryption Using GUI

This GUI version allows the user to:

- Enter any text message they want to encrypt or decrypt.

- Specify the shift value (number of positions to shift letters).

- Click a button to encrypt the text, which is then displayed in the encrypted output box.

- Automatically see the decrypted text for verification.
